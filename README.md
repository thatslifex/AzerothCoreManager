## ✨ Features

### 🖥️ Server Management

| Page | Description |
|---|---|
| **Dashboard** | Live status polling (authserver, worldserver, MySQL), player count, update check |
| **Server Control** | Start / Stop / Restart authserver + worldserver, live console output, uptime tracking |
| **GM Console** | SOAP command interface, 20+ categorized commands, command history, favorites |

### 🔧 Setup & Configuration

| Page | Description |
|---|---|
| **Setup Pipeline** | 6-step guided installation: system check → prerequisites → source clone → CMake build → deploy → client data |
| **Config Editor** | Visual `.conf` editor (worldserver.conf / authserver.conf) with search and atomic save |
| **Module Manager** | GitHub catalog search (`topic:azerothcore-module`), install / update / remove with CMake rebuild |

### 🗄️ Database & Community

| Page | Description |
|---|---|
| **Database** | Accounts, Characters, Bans, SQL Query Editor, Create Account dialog |
| **Accounts** | Player activity overview, mass account creation (up to 100 at once), GM audit log |
| **Backup Manager** | `mysqldump` all 3 databases, restore from backup, backup history |

### ⚙️ System

| Page | Description |
|---|---|
| **Settings** | Server paths, MySQL / SOAP config, theme / language / background, export / import JSON |
| **Log Viewer** | File tail, text filter, auto-refresh (5s), auto-discover log files |
| **Self-Update** | Auto-check on startup, download progress bar, one-click install |

### 🌍 Localization

**6 languages** — switch instantly in Settings:

- 🇬🇧 English
- 🇩🇪 Deutsch
- 🇫🇷 Français
- 🇪🇸 Español
- 🇨🇳 中文
- 🇷🇺 Русский

Navigation, status bar, and all UI strings are fully localized.

## 🚀 Quick Start

1. **Download** the [latest installer](https://github.com/thatslifex/AzerothCoreManager/releases/latest)
2. **Run** `AzerothCoreManager-Setup-1.0.0.exe`
3. **Launch** AzerothCore Manager from the Start Menu
4. The **First Run Wizard** guides you through the 6-step setup
5. Once setup is complete, use the Dashboard to monitor your server
